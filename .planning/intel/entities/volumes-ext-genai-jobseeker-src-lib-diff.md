# diff.tsx

## Purpose

Renders an inline word-level diff between two strings, highlighting additions in green and removals in red with strikethrough. Exists to visually compare original and rewritten text in the UI.

## Exports

- `InlineDiff` — React component that accepts `original` and `rewritten` strings and renders a styled inline diff
