# layout.tsx

## Purpose

Root layout component for the Next.js app, wrapping all pages with the Inter font and base HTML structure. Defines the global metadata for the application.

## Exports

- `metadata` — Next.js Metadata object with app title and description
- `RootLayout` (default) — Root layout component that wraps all page children with `<html>` and `<body>` tags

## Dependencies

- next/font/google (Inter)
- [[globals.css]]
