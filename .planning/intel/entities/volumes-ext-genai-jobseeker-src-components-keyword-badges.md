# keyword-badges.tsx

## Purpose

Displays a card listing keyword gaps between a job description and the user's resume. Renders each missing keyword as an outline badge, prompting the user to add relevant ones where truthful.

## Exports

- `KeywordBadges` — React component accepting a `keywords: string[]` prop; renders a Card with badges for each gap, or an empty-state message when none exist.

## Dependencies

- [[card]] — Card, CardContent, CardDescription, CardHeader, CardTitle
- [[badge]] — Badge
